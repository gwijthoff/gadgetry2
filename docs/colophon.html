Colophon

Site’s bones are Jekyll (a static site generator), fleshed out in Tachyons (a CSS library), as mashed up in jekyons.

I compile my research data in a Google spreadsheet, then export to JSON using an add-on called ExportSheetData. This site then pulls from the JSON data each time it’s generated using the jekyll-get plugin.

My use of Font Awesome icons was inspired by Otto Neurath’s 1920s ISOTYPE.

Generated using Jekyll, Tachyons, and JSON data by Grant Wythoff.
